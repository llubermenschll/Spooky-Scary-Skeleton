# Spooky-Scary-Skeleton
Animazione: scheletro danzante

Questo progetto è stato interamente scritto in linguaggio C ed è stato sviluppato in ambiente Gnu/Linux. Eventuali problemi di funzionamento su Windows possono
essere risolti modificando l'argomento della funzione system nel codice sorgente e verificando che, in fase di esecuzione, sia gestita correttamente la
visualizzazione dei simboli stampati.
